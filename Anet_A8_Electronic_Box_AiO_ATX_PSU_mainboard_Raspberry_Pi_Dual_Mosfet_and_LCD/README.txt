# Summary

Update 09/2018 : add the support for the Pi Camera. You need to fix it on the first screw in the front of the bed.

<hr>


Enough of messy wires. This is an All In One box for Anet A8 like for the Creality CR-10.

It includes :
- ATX power Supply
- Anet A8 mainboard
- dual Mosfet
- Raspberry Pi 3
- LCD
- cooling for the electronics boards
- spool holder (need 2 bearings)
- bases for under cooling the PSU
- some cable management inside the box

This is my first public version, thanks for support.
