# Summary

Raspberry Pi camera case is GoPro and Modular Mounting System compatible. There are two different backs; BACK1 will mate cleanly with the mounting tabs and presents a more finished look, but it requires supports, BACK2 prints without supports, but the mounting tabs will overhang the back of the case. The case is fully @dcgrigsby's design, I just added the GoPro tabs.

In the second image, BACK1 is on the left, and BACK2 is on the right.

This is for Raspberry Pi Camera v2
