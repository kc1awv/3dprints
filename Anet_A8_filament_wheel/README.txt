# Summary

A wheel to help the filament turn smoother on the standard Anet a8 filament holder. I designed this to be filament cheap, and fully printable. I designed this for Hatchbox filaments, but it will work with any filament with a hole 50-65mm (2.1-2.5in). Requires two wheels.

# Print Settings

Printer: Anet a8
Rafts: Doesn't Matter
Supports: No
Infill: 10
