**If you print this thing and find it useful, please consider hitting that "Tip Designer" 
 button and help support more designs.**


I really liked the original design by Leo_N ([www.thingiverse.com/thing:1857991](www.thingiverse.com/thing:1857991)), but to add additional stability thought it would be nice to have holes already in the model that I could screw it down to my base.
