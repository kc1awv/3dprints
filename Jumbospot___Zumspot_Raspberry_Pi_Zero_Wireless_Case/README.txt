# Summary

This case is to suit the Jumbospot Pi hat for the Raspberry Pi Zero W found on eBay which seems to be a clone of the Zumspot.

Created in openSCAD if you need to customise.

1. Print the 3 parts
2. Insert the Pi into the lower piece
3. Press the insert onto the Pi
4. Insert the Jumbospot into the upper piece
5. Press upper and lower together

Optionally you can add a heatsink on top.
