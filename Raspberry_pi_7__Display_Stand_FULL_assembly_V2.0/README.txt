# Summary

This is updated (18-07-15)  stand assembly for the raspberry pi 7 inch display. You can mount it on your 2020 extruded 3d printer frame. 
Printed in PLA. Good for your octoPi touch screen set up.
Fit the eom raspberry pi 7 inch touchscreen display,
This was design with using the Official Raspberry Pi 7" Touch Screen Reference Model from user clough42.
