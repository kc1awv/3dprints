# Summary

This thing is a belt holder for the x axis of the Anet A8 (May 2017). I had to design a belt holder which fits for my GT2 belts. The other belt holders I've found around thingiverse didn't fit well.

You also need minimum 4x M3x22mm screws to mount the holder to the slider.
